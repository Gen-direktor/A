Введение в тему

Веб-разработка — это процесс создания веб-сайтов и веб-приложений. Она включает в себя:

  • Создание структуры сайта
  • Оформление внешнего вида
  • Программирование функционала
  • Оптимизацию для пользователей

Сайты нужны для:

  • Представления компании в интернете
  • Продажи товаров и услуг
  • Информационного обмена
  • Развлечения
  • Обучения

Основные компоненты сайта:

  • HTML — структура контента
  • CSS — стили и оформление
  • JavaScript — интерактивность
  • Серверные языки — обработка данных

Про HTML

HTML (HyperText Markup Language) — это язык разметки гипертекста, который:

  • Определяет структуру веб-страницы
  • Создает основу для контента
  • Обеспечивает семантическую разметку

Базовые теги HTML:

  • Заголовки: <h1> до <h6>
  • Абзацы: <p>
  • Списки: <ul>, <ol>, <li>
  • Ссылки: <a href="url">
  • Изображения: <img src="путь">

Пример простой HTML-страницы:

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Пример страницы</title>
</head>
<body>
    <h1>Привет, мир!</h1>
    <p>Это пример HTML-страницы.</p>
</body>
</html>

Про CSS

CSS (Cascading Style Sheets) — это язык стилей, который:

  • Управляет внешним видом HTML-элементов
  • Определяет макет страницы
  • Создает адаптивный дизайн

Основные свойства CSS:

  • Цвета: color, background-color
  • Размеры: width, height
  • Позиционирование: position, float
  • Отступы: margin, padding

    • Азбука CSS

    Основные концепции и свойства CSS:

    Базовые селекторы

    • Универсальный селектор: *
    • Селектор элемента: div, p, h1
    • Селектор класса: .class-name
    • Селектор идентификатора: #id-name

    Свойства оформления

    • Цвета:
      • color — цвет текста
      • background-color — цвет фона
    • Шрифты:
      • font-family — семейство шрифтов
      • font-size — размер шрифта
      • font-weight — жирность

    Свойства позиционирования

    • Размеры:
      • width — ширина
      • height — высота
    • Отступы:
      • margin — внешние отступы
      • padding — внутренние отступы
    • Позиционирование:
      • position — тип позиционирования
      • float — обтекание
      • display — тип отображения

    Псевдоклассы и псевдоэлементы

    • Псевдоклассы:
      • :hover — при наведении
      • :focus — при фокусе
      • — посещенная ссылка
    • Псевдоэлементы:
      • ::before — вставка перед элементом
      • ::after — вставка после элемента

    Пример стилей

    .container {
    width: 80%;
    margin: 0 auto;
}

h1 {
    color: #333;
    text-align: center;
}

p {
    line-height: 1.6;
    margin: 20px 0;
}

a:hover {
    text-decoration: none;
    color: #007bff;
}

    Азбука HTML

    Давайте разберем основные теги HTML:

    Базовая структура документа

    • <!DOCTYPE html> — объявление типа документа
    • <html> — корневой элемент
    • <head> — содержит метаинформацию
    • <body> — видимое содержимое страницы

    Форматирование текста

    • <p> — абзац
    • <h1>-<h6> — заголовки разного уровня
    • <strong> — важный текст (жирный)
    • <em> — акцент (курсив)
    • <br> — перенос строки
    • <hr> — горизонтальная линия

    Списки

    • <ul> — маркированный список
    • <ol> — нумерованный список
    • <li> — элемент списка

    Ссылки и медиа

    • <a href="url"> — гиперссылка
    • <img src="путь"> — изображение
    • <video> — видео
    • <audio> — аудио

    Таблицы

    • <table> — таблица
    • <tr> — строка таблицы
    • <th> — заголовок столбца
    • <td> — ячейка таблицы

    Формы

    • <form> — форма
    • <input> — поле ввода
    • <textarea> — многострочное поле
    • <select> — выпадающий список

    Практический пример создания простой веб-страницы

    Давайте создадим простую веб-страницу с использованием HTML и CSS.

    Шаг 1: Создаем базовую структуру HTML

    <!DOCTYPE html>
<html lang="ru">
<head>
 <meta charset="UTF-8">
 <meta name="viewport" content="width=device-width, initial-scale=1.0">
 <title>Простая веб-страница</title>
 <link rel="stylesheet" href="css/styles.css">
</head>
<body>
 <header>
 <h1>Добро пожаловать на мой сайт</h1>
 </header>
 
 <main>
 <p>Это пример простой веб-страницы, созданной с помощью HTML и CSS.</p>
 
 <h2>Наши услуги</h2>
 <ul>
 <li>Веб-разработка</li>
 <li>Дизайн</li>
 <li>SEO-оптимизация</li>
 </ul>
 </main>
 
 <footer>
 <p>© 2025 Мой сайт</p>
 </footer>
</body>
</html>

    Шаг 2: Добавляем стили CSS

    Создаем файл styles.css со следующим содержимым:

    body {
 margin: 0;
 font-family: Arial, sans-serif;
}

header {
 background-color: #f8f9fa;
 padding: 20px;
 text-align: center;
}

main {
 padding: 20px;
}

ul {
 list-style-type: none;
 padding: 0;
}

li {
 margin: 10px 0;
}

footer {
 background-color: #343a40;
 color: white;
 text-align: center;
 padding: 15px 0;
}

    Результат

    После подключения всех файлов у нас получится простая, но функциональная веб-страница с:

    • Заголовочной частью
    • Основным контентом
    • Подвалом с копирайтом
    • Базовой стилизацией

    Этот пример можно использовать как основу для создания более сложных проектов.
